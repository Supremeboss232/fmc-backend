Developer: API integration (examples)

The page will fetch live price and recent trades from your backend at http://localhost:4000/

// Example fetch to get live price (replace endpoints as needed)
fetch('http://localhost:4000/price/latest')
  .then(r => r.json())
  .then(data => console.log('price', data))
  .catch(e => console.error(e));

// Example fetch to list recent trades
fetch('http://localhost:4000/trades/recent')
  .then(r => r.json())
  .then(data => console.log('trades', data))
  .catch(e => console.error(e));
